# Babel 图书馆

**让学术图书馆走向公共空间**

欢迎来到 **Babel 图书馆** —— 一个连接新移民、普通读者与研究者的双语平台。  

## 模块
- 📚 **图书馆必备**：办卡指南、电子资源、大温图书馆使用技巧。  
- 📖 **阅读与书单**：以学术主题为线索，匹配公共馆藏的可读书目。  
- 👥 **活动与社群**：每周精选活动与 Babel 读书会。  
- 🛠 **信息摄取指南**：从 Libby 到 JSTOR、Zotero 的实用教程。  
- 🔗 **学术 ↔ 公共 桥梁**：把学术馆藏与研究方法转译为大众可用的阅读路径。  
- 📊 **数据看板**：可视化外展与触达情况。
